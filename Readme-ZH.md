# Maibook

Maibook 是一款面向学术论文阅读的本地桌面应用，基于 Tauri 构建。它提供
高性能 PDF 阅读器、AI 驱动的标注与翻译、本地论文库管理，以及可扩展的
AI 技能体系（Skill Hub）。

## 项目愿景

Maibook 致力于成为学术工作流的全平台一体化助手：

- **智能发现** — 输入研究主题，自动检索并导入自定义数量的相关论文，一键收入论文库。
- **阅读自动化** — 对导入的论文自动完成摘要、翻译和标签分类，让你无需打开 PDF 就能掌握核心内容。
- **组会助手** — 通过"组会"功能，自动将本周阅读的论文整理为周报、PPT 和对应演讲稿，组会准备从数小时缩短到几分钟。

从论文检索到组会汇报，Maibook 闭环学术工作的每一环。

> 本项目受 Moonlight Chrome 扩展启发，进行了自主架构设计与功能创新。

## 功能

### PDF 阅读器

- 基于 pdf.js 深度定制，流畅渲染 PDF 文档
- 页面导航、缩放、多布局模式（单页 / 双页 / 连续滚动）
- 全文搜索、大纲提取、书签
- 拖拽打开 / 文件菜单 / URL 打开

### AI 引擎

- 兼容 OpenAI API 格式（支持 DeepSeek、通义千问等第三方提供商）
- 选中文本：AI 解释、翻译、摘要
- 图片 / 表格 / 公式智能分析
- AI 对话侧边栏（多轮上下文）

### 标注系统

- 文本高亮（预设色 + 自定义色）
- 批注评论（关联论文位置）
- 画笔绘制（颜色 / 粗细 / 可擦除）
- 标注索引面板（筛选 + 跳转）

### 翻译

- 内联翻译：选中文本弹出对照气泡
- 整页翻译：半透明层叠模式

### 论文库管理

- 论文列表 / 网格视图
- 拖拽导入 / 对话框导入 / 文件夹监视
- AI 元数据提取（标题、作者、期刊、DOI）
- 标签分类、分组管理、阅读状态追踪
- BibTeX / RIS 引用导出

### AI Skill Hub（创新功能）

- 将 AI 能力抽象为可扩展的技能体系
- 可配置 Hub URL，自动获取技能市场目录
- 一键安装 / 卸载技能
- AI 引擎根据场景自动编排已安装技能

## 技术栈

| 层级 | 选型 | 说明 |
|------|------|------|
| 桌面框架 | **Tauri** | 跨平台桌面框架，Rust 核心 + Web 前端 |
| 前端框架 | **React** | 构建 UI 组件和交互界面 |
| PDF 引擎 | **pdf.js** | 深度定制的 PDF 渲染引擎 |
| 本地存储 | **SQLite** | 结构化数据存储 |
| AI 接口 | **OpenAI 兼容 API** | 支持 OpenAI、DeepSeek、通义千问等 |
| 状态管理 | **Zustand** | 轻量响应式状态管理 |
| 国际化 | **i18next** | 中英双语 key-value 模式 |
| 打包格式 | **ZIP（.maibook）** | 数据导出 / 导入便携格式 |

`

## 开始使用

### 前置要求

- [Node.js](https://nodejs.org/) >= 18
- [pnpm](https://pnpm.io/)
- [Rust](https://www.rust-lang.org/)（用于 Tauri 构建）

### 安装与运行

`ash
# 安装前端依赖
pnpm install

# 启动开发服务器（含 Tauri 窗口）
pnpm tauri dev
`

### 构建

`ash
pnpm tauri build
`
